# 📌 Freelance-App

Aplicación web desarrollada para ayudar a freelancers a organizar, controlar y dar seguimiento a sus proyectos, clientes y tareas de manera eficiente.

---

## 🚀 Descripción

**Freelance-App** es un sistema enfocado en la gestión de trabajo freelance, permitiendo al usuario tener control total sobre:

- Clientes
- Proyectos
- Tareas
- Tiempos de ejecución

El objetivo principal es mejorar la organización, productividad y toma de decisiones basadas en el tiempo invertido en cada actividad.

---

## 🛠️ Tecnologías utilizadas

- **Backend:** PHP (Yii Framework 1.1 - MVC)
- **Base de datos:** MySQL
- **Arquitectura:** MVC (Modelo - Vista - Controlador)
- **Control de acceso:** Usuarios, Roles y Permisos

---

## 📂 Estructura del sistema

El sistema está dividido en los siguientes módulos:

### 👤 Usuarios
- Gestión de usuarios del sistema
- Control de acceso mediante roles y permisos

### 🧑‍💼 Clientes
- Registro de clientes
- Base para asignación de proyectos

### 📁 Proyectos
- Creación de proyectos asociados a clientes
- Punto central del sistema

### ✅ Tareas
- Definición de actividades dentro de un proyecto
- Descripción clara de lo que se debe realizar

### ⏱️ Tiempos
- Registro del tiempo invertido por tarea
- Permite analizar:
  - Cuánto tiempo toma cada tarea
  - Identificar cuellos de botella
  - Mejorar estimaciones futuras

---

## 🔮 Funcionalidades futuras

El sistema está diseñado para escalar con los siguientes módulos:

### 💰 Cotizaciones
- Generación de presupuestos
- Estimación de tiempos y costos por proyecto
- Definición de tareas dentro de la cotización

### 📊 Finanzas
- Control de ingresos y egresos
- Análisis financiero del trabajo freelance
